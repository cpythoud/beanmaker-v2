NOUVEL EXPORT EXCEL
===================

Based on displayed columns on page, with option to include everything

MasterTable
-----------

Reactivate display of Excel export icon

Add:
- class tb-filter-row in filters <tr>
- class tb-title-row  in titles  <tr>

In titles, add properties data-fieldname and data-datatype on the <th>, with possible values for the latter:
- str (default)  -> String, Bean references
- int            -> Integer, Long
- float          -> (no direct correspondance, but should be available for ad hoc cells)
- bool           -> Boolean
- date           -> java.sql.Date
- time           -> java.sql.Time
- timestamp      -> java.sql.Timestamp
- money          -> Money (class from runtime)

In cells:

- optional data-spreadsheet, in case the data to appear on the spreadsheet is different than what's shown on screen

-----------------

- add Excel op to OperationBaseServlet

- public ExcelExporter getExcelExporter() { return DEFAULT_EXCEL_EXPORTER } in OperationBaseServlet, can be overriden in BaseServlet
